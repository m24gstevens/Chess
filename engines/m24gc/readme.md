# M24GC

A UCI engine written in C

## Features
Board Representation
- Bitboards
- 8x8 board

Search
- Iterative Deepening
- Alpha-Beta
- Move-Ordering
- Transposition table
- Null Move heuristic

Evaluation
- Piece square tables
- Tapered evaluation
