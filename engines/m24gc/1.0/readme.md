# M24GC

## Features in 1.0
Board Representation
- Bitboards
- 8x8 board
- Legal move generation
- Make-Unmake

Search
- Iterative Deepening
- Alpha-Beta
- Move-Ordering
- Transposition table
- Null Move heuristic

Evaluation
- Piece square tables
- Pawn structure (isolated, doubled, backward, passed)
- King safety (open files, pawn shield)
- Mobility
